# Skills / Reusable Procedures

## Verify ODrive Setup
1. Confirm the board is visible.
2. Check calibration status.
3. Clear errors.
4. Set torque control and passthrough input mode.
5. Test closed loop.

## Run Single-Axis CTC Test
1. Connect GUI.
2. Set offset.
3. Enable torque.
4. Use low torque limits.
5. Verify desired motion tracking.

## Inspect TWAI Bridge
1. Verify CAN speed and wiring.
2. Confirm frame format.
3. Test transmit/receive.
4. Check bus alerts.
5. Recover from bus-off if needed.
